## Lesson2 :: Реализация рекурсивной функции в Node.js

### Задание:
Напиши программу, которая использует рекурсию для вычисления суммы цифр числа

пример работы программы:
Введите натуральное число: 1234
Сумма цифр числа 1234 = 10


### Требования к программе:
   - программа должна содержать рекурсивную функцию sumOfDigits(n), которая принимает натуральное число n и возвращает сумму его цифр.
   - функция должна корректно обрабатывать как однозначные, так и многозначные числа.
   - программа должна запрашивать число у пользователя через readline
   - вывести результат в консоль.

### Алгоритм:
1) Если n однозначное (n < 10), просто вернуть n.
2) В остальных случаях:
   - Найти последнюю цифру числа:
      ```js
       n % 10
       ```
   - найти оставшуюся часть числа (см подсказку)
   - вызвать sumOfDigits для оставшейся части
   - Вернуть сумму последней цифры и суммы оставшихся цифр
3) вывести результат
